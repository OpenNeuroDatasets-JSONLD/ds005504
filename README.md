Note to investigators: If you would like to use this dataset, we kindly ask that you inform Charles Nelson (charles.nelson@childrens.harvard.edu) of your research question prior to or shortly following download. Doing so will allow us to maintain an inventory of this dataset's uses in the scientific community and offer investigators the opportunity to request additional non-MRI measures from this cohort, which they would do through this link: https://redcap.tch.harvard.edu/redcap_edc/surveys/?s=XDMFPCN8F9. As this openneuro dataset is publicly available, communication with Dr. Nelson will have no bearing on your access to it.
